# K值计算逻辑说明

**版本**: v2.0
**日期**: 2026-05-21
**适用范围**: 朔黄铁路跨度32m梁检定承载系数计算

---

## 一、概述

### 1.1 K值定义

**检定承载系数K**：衡量桥梁在当前状态下承载能力是否满足运营要求的无量纲指标。

### 1.2 计算原则

- 取多个检算项目的最不利值作为整体检定承载系数
- K = min{K1, K2, K3, K4, K5}

### 1.3 检算项目

共5项分项检定承载系数：

| 编号 | 检算项目 | 检算截面 | 控制指标 |
|------|----------|----------|----------|
| K1 | 正截面抗弯强度 | 跨中截面 | 弯矩M |
| K2 | 正截面抗裂性 | 跨中截面 | 下缘正应力σ |
| K3 | 正截面应力 | 跨中截面 | 下缘正应力σ |
| K4 | 斜截面剪应力 | L/8截面 | 剪应力τ |
| K5 | 斜截面抗裂性 | L/8截面 | 主拉应力σtp |

## 二、计算流程

```
┌─────────────────────────────────────────────────────────────────┐
│                        K值计算流程                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  步骤1: 确定梁型                                                 │
│    └─→ 根据孔跨式样匹配固定参数（内力、强度等）                    │
│                                                                 │
│  步骤2: 用户输入参数                                             │
│    └─→ 道砟厚度t、偏心值e、冲击系数1+μ、曲线半径R                 │
│                                                                 │
│  步骤3: 查表获取偏载系数                                         │
│    └─→ 根据e、R、梁体类型查表得到ηM、ηQ                           │
│                                                                 │
│  步骤4: 计算主力组合内力/应力                                     │
│    ├─→ 跨中弯矩: M = Mq0+Mq1+Mq2+t/10·Mq3+(1+μ)·ηM·ML           │
│    ├─→ 跨中应力: σ = σq0+σq1+σq2+t/10·σq3+(1+μ)·ηM·σL           │
│    └─→ L/8剪力: τ = τq0+τq1+τq2+t/10·τq3+(1+μ)·ηM·τL           │
│                                                                 │
│  步骤5: 计算分项检定承载系数                                     │
│    ├─→ K1: 正截面抗弯强度                                        │
│    ├─→ K2: 正截面抗裂性                                          │
│    ├─→ K3: 正截面应力                                            │
│    ├─→ K4: 斜截面剪应力                                          │
│    └─→ K5: 斜截面抗裂性                                          │
│                                                                 │
│  步骤6: 确定最终K值                                              │
│    └─→ K = min{K1, K2, K3, K4, K5}                              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 三、主力组合计算

### 3.1 跨中截面弯矩

$$M = M_{q0} + M_{q1} + M_{q2} + \frac{t}{10} \times M_{q3} + (1+\mu) \times \eta_M \times M_L$$

### 3.2 跨中截面下缘正应力

$$\sigma = \sigma_{q0} + \sigma_{q1} + \sigma_{q2} + \frac{t}{10} \times \sigma_{q3} + (1+\mu) \times \eta_M \times \sigma_L$$

### 3.3 L/8截面剪应力

$$\tau = \tau_{q0} + \tau_{q1} + \tau_{q2} + \frac{t}{10} \times \tau_{q3} + (1+\mu) \times \eta_M \times \tau_L$$

### 3.4 参数说明

| 参数 | 含义 | 单位 | 来源 |
|------|------|------|------|
| M, σ, τ | 主力组合内力/应力 | kN·m / MPa | 计算得到 |
| M_q0, σ_q0, τ_q0 | 梁体自重效应 | kN·m / MPa | 固定参数 |
| M_q1, σ_q1, τ_q1 | 设计二期恒载效应 | kN·m / MPa | 固定参数（按梁型） |
| M_q2, σ_q2, τ_q2 | 线桥改造增加恒载效应 | kN·m / MPa | 固定参数 |
| t | 道砟超厚 | cm | **用户输入** |
| M_q3, σ_q3, τ_q3 | 道砟每10cm恒载效应 | kN·m / MPa | 固定参数 |
| 1+μ | 冲击系数 | - | 用户输入（默认1.1935） |
| η_M, η_Q | 偏载系数 | - | **查表得到** |
| M_L, σ_L, τ_L | 静活载效应 | kN·m / MPa | 固定参数（中-活载） |

### 4.4 插值方法

当输入的偏心值e不在表格中时，采用**线性插值**：

$$\eta = \eta_1 + \frac{e - e_1}{e_2 - e_1} \times (\eta_2 - \eta_1)$$

其中：e1 < e < e2，η1、η2为对应的偏载系数

## 五、分项检定承载系数计算

### 5.1 K1：正截面抗弯强度（跨中截面）

**计算公式**：

$$K_1 = \frac{Z_{1M} \cdot M_p / K_w - (M_{q0} + M_{q1} + M_{q2} + \frac{t}{10} \times M_{q3})}{(1+\mu) \times \eta \times M_L^{中-活载}}$$

**参数说明**：

| 参数 | 含义 | 取值 |
|------|------|------|
| Z_1M | 抗弯强度修正系数 | 用户输入（默认1.00） |
| M_p | 正截面抗弯强度 | 固定（直线梁29752 / 曲线梁33647 kN·m） |
| K_w | 强度安全系数容许值 | 固定（2.0） |
| M_L^{中-活载} | 中-活载静活载弯矩 | 固定（6294.5 kN·m） |

**设计要求**：主力组合强度安全系数 Kw = Mp / M ≥ 2

### 5.2 K2：正截面抗裂性（跨中截面）

**计算公式**：

$$K_2 = \frac{Z_{2M} \cdot Z_{4Y} \cdot \sigma_{pc} + \gamma \cdot Z_{3H} \cdot f_{ct} / K - Z_{2M} \cdot (\sigma_{q0} + \sigma_{q1} + \sigma_{q2} + \frac{t}{10} \times \sigma_{q3})}{Z_{2M} \times (1+\mu) \times \eta \times \sigma_L^{中-活载}}$$

**参数说明**：

| 参数 | 含义 | 取值 |
|------|------|------|
| Z_2M | 截面弯曲正应力修正系数 | 用户输入（默认1.00） |
| Z_4Y | 永存预应力修正系数 | 用户输入（默认1.00） |
| Z_3H | 混凝土强度修正系数 | 用户输入（默认1.00） |
| σ_pc | 预应力引起的下缘正应力 | 固定（直线梁20.956 / 曲线梁23.995 MPa） |
| γ | 混凝土塑性修正系数 | 固定（直线梁1.3234 / 曲线梁1.3228） |
| f_ct | 混凝土抗拉极限强度 | 固定（3.02 MPa） |
| K | 正截面抗裂安全系数容许值 | 固定（1.2） |

**设计要求**：主力组合抗裂安全系数 K = (σpc + γ·fct) / σ ≥ 1.2

### 5.3 K3：正截面应力（跨中截面）

**计算公式**：

$$K_3 = \frac{Z_{2M} \cdot Z_{4Y} \cdot \sigma_{pc} - Z_{2M} \cdot (\sigma_{q0} + \sigma_{q1} + \sigma_{q2} + \frac{t}{10} \times \sigma_{q3})}{Z_{2M} \times (1+\mu) \times \eta \times \sigma_L^{中-活载}}$$

**参数说明**：

| 参数 | 含义 | 取值 |
|------|------|------|
| σ_pc | 预应力引起的下缘正应力 | 固定（直线梁20.956 / 曲线梁23.995 MPa） |

**设计要求**：跨中截面下缘应力 σct = σpc - σ ≥ 0

### 5.4 K4：斜截面剪应力（L/8截面）

**计算公式**：

$$K_4 = \frac{(Z_{2Q} \cdot Z_{4Y} \cdot \tau_{pc} + Z_{3H} \times 0.17 f_c) - Z_{2Q} \cdot (\tau_{q0} + \tau_{q1} + \tau_{q2} + \frac{t}{10} \times \tau_{q3})}{Z_{2Q} \times (1+\mu) \times \eta \times \tau_L^{中-活载}}$$

**参数说明**：

| 参数 | 含义 | 取值 |
|------|------|------|
| Z_2Q | 截面弯曲剪应力修正系数 | 用户输入（默认1.00） |
| τ_pc | 预应力引起的剪应力 | 固定（按梁型查表） |
| f_c | 混凝土抗压极限强度 | 固定（32.10 MPa） |

**设计要求**：斜截面剪应力 τc = τ - τpc ≤ 0.17·fc

### 5.5 K5：斜截面抗裂性（L/8截面）

**计算公式**：

K5通过以下方程组求解，应满足 |σtp| ≤ Z3H·fct：

$$\sigma_c = Z_{2M} \cdot Z_{4Y} \cdot \sigma_{pc} - K \cdot [Z_{2M} \cdot (\sigma_{q0} + \sigma_{q1} + \sigma_{q2} + \frac{t}{10} \times \sigma_{q3}) + K_5 \cdot Z_{2M} \cdot (1+\mu) \cdot \eta \cdot \sigma_L^{中-活载}]$$

$$\tau_c = K \cdot [Z_{2Q} \cdot (\tau_{q0} + \tau_{q1} + \tau_{q2} + \frac{t}{10} \times \tau_{q3}) + K_5 \cdot Z_{2Q} \cdot (1+\mu) \cdot \eta \cdot \tau_L^{中-活载}] - Z_{2Q} \cdot Z_{4Y} \cdot \tau_{pc}$$

$$\sigma_{tp} = \frac{\sigma_c}{2} - \sqrt{\left(\frac{\sigma_c}{2}\right)^2 + \tau_c^2}$$

其中K为正截面抗裂安全系数容许值（取1.2）。

**设计要求**：|σtp| ≤ Z3H·fct，其中σtp计算值为负值，取绝对值。

## 六、最终K值确定

### 6.1 计算公式

$$K = \min\{K_1, K_2, K_3, K_4, K_5\}$$

### 6.2 判定标准

| K值范围 | 判定结果 | 处理措施 |
|---------|----------|----------|
| K ≥ 1.0 | 满足"中-活载"要求 | 正常运营 |
| K < 1.0 | 不满足"中-活载"要求 | 需进一步计算运行列车活载系数Q |

### 6.3 K < 1.0时的补充计算

当K < 1时，需进一步计算运行列车（C80编组、KM98编组）的活载系数Q，判断是否满足实际运营列车要求：

$$Q_{C80} = \max\left(\frac{M_L^{C80}}{M_L^{中-活载}}, \frac{V_L^{C80}}{V_L^{中-活载}}\right)$$

$$Q_{KM98} = \max\left(\frac{M_L^{KM98}}{M_L^{中-活载}}, \frac{V_L^{KM98}}{V_L^{中-活载}}\right)$$

若 Q < K，则该运行列车满足要求。

## 九、计算示例

### 9.1 输入条件

- 桥梁：清水河特大桥
- 梁型：专桥2059，直线梁
- 道砟超厚：t = 5 cm
- 线梁偏心：e = 70 mm
- 冲击系数：1+μ = 1.1935

### 9.2 计算过程

**步骤1：查表获取偏载系数**
- e = 70mm，直线梁
- 查表得：η_M = 1.042

**步骤2：计算主力组合弯矩**
```
M = 4248.0 + 2367.2 + 811.9 + (5/10)×497.7 + 1.1935×1.042×6294.5
M = 4248.0 + 2367.2 + 811.9 + 248.85 + 7827.3
M = 15503.25 kN·m
```

**步骤3：计算K1**
```
K1 = [1.00 × 29752 / 2.0 - (4248.0 + 2367.2 + 811.9 + 248.85)] / [1.1935 × 1.042 × 6294.5]
K1 = [14876 - 7675.95] / 7827.3
K1 = 7200.05 / 7827.3
K1 = 0.92
```

**步骤4：计算K2、K3、K4、K5**
- 按相应公式计算（需查取应力参数表）

**步骤5：确定最终K值**
```
K = min{K1, K2, K3, K4, K5}
```

### 9.3 结论

根据最终K值判定：
- K ≥ 1.0：满足"中-活载"运营要求
- K < 1.0：需进一步计算C80、KM98编组活载系数Q
