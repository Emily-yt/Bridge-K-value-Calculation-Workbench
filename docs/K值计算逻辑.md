# K值计算逻辑说明

**版本**: v1.0  
**日期**: 2026-05-10  
**适用范围**: 朔黄铁路跨度32m梁检定承载系数计算

---

## 一、概述

### 1.1 K值定义

**检定承载系数K**：衡量桥梁在当前状态下承载能力是否满足运营要求的无量纲指标。

| K值范围 | 判定结果 |
|---------|----------|
| K ≥ 2.0 | 设计安全储备满足规范要求 |
| K ≥ 1.0 | 满足运营要求，可正常运行 |
| K < 1.0 | 不满足运营要求，需限速或加固 |

### 1.2 计算原则

- 取多个检算项目的最不利值作为整体检定承载系数
- K = min{K1, K2, K3, K4}

---

## 二、计算流程

```
┌─────────────────────────────────────────────────────────────────┐
│                        K值计算流程                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  步骤1: 确定梁型                                                 │
│    └─→ 根据孔跨式样匹配固定参数（内力、强度等）                    │
│                                                                 │
│  步骤2: 用户输入参数                                             │
│    └─→ 道砟厚度t、偏心值e、冲击系数1+μ、曲线半径R                 │
│                                                                 │
│  步骤3: 查表获取偏载系数                                         │
│    └─→ 根据e、R、梁体类型查表得到η                               │
│                                                                 │
│  步骤4: 计算主力组合弯矩M                                        │
│    └─→ M = M_q0 + M_q1 + M_q2 + (t/10)×M_q3 + (1+μ)×η×M_L       │
│                                                                 │
│  步骤5: 计算分项检定承载系数                                     │
│    ├─→ K1: 正截面抗弯强度                                        │
│    ├─→ K2: 正截面抗裂性                                          │
│    ├─→ K3: 正截面应力                                            │
│    └─→ K4: 斜截面应力                                            │
│                                                                 │
│  步骤6: 确定最终K值                                              │
│    └─→ K = min{K1, K2, K3, K4}                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## 三、主力组合弯矩计算

### 3.1 计算公式

$$M = M_{q0} + M_{q1} + M_{q2} + \frac{t}{10} \times M_{q3} + (1+\mu) \times \eta \times M_L$$

### 3.2 参数说明

| 参数 | 含义 | 单位 | 来源 |
|------|------|------|------|
| M | 主力组合弯矩 | kN·m | 计算得到 |
| M_q0 | 梁体自重弯矩 | kN·m | 固定参数 |
| M_q1 | 设计二期恒载弯矩 | kN·m | 固定参数（按梁型） |
| M_q2 | 线桥改造增加恒载弯矩 | kN·m | 固定参数 |
| t | 道砟超厚 | cm | **用户输入** |
| M_q3 | 道砟每10cm恒载弯矩 | kN·m | 固定参数 |
| 1+μ | 冲击系数 | - | 用户输入（默认1.1935） |
| η | 偏载系数 | - | **查表得到** |
| M_L | 静活载弯矩 | kN·m | 固定参数 |

### 3.3 固定参数取值（专桥2059，32m梁）

#### 跨中截面弯矩

| 参数 | 直线梁 | 曲线外梁 | 曲线内梁 |
|------|--------|----------|----------|
| M_q0 | 4248 | 4248 | 4248 |
| M_q1 | 2367.2 | 3365.2 | 2482.3 |
| M_q2 | 811.9 | 811.9 | 811.9 |
| M_q3（每10cm） | 497.7 | 497.7 | 497.7 |
| M_L（中-活载） | 7512.8 | 7512.8 | 7512.8 |

#### L/8截面内力

| 参数 | M_max (kN·m) | Q_min (kN) |
|------|--------------|------------|
| M_q0 | 1857.6 | -398.3 |
| M_q1（直线梁） | 1035.2 | -222 |

- **曲线半径 R**：曲线梁时提供，单位m

### 4.4 插值方法

当输入的偏心值e不在表格中时，采用**线性插值**：

$$\eta = \eta_1 + \frac{e - e_1}{e_2 - e_1} \times (\eta_2 - \eta_1)$$

其中：e1 < e < e2，η1、η2为对应的偏载系数

## 五、分项检定承载系数计算

### 5.1 K1：正截面抗弯强度

**计算公式**：

$$K_1 = \frac{Z_{1M} \times M_p / K^w - (M_{q0} + M_{q1} + M_{q2} + \frac{t}{10} \times M_{q3})}{(1+\mu) \times \eta \times M_L^{中-活载}}$$

**参数说明**：

| 参数 | 含义 | 取值 |
|------|------|------|
| Z_1M | 抗弯强度修正系数 | 用户输入（默认1.00） |
| M_p | 正截面抗弯强度 | 固定（直线梁29752 / 曲线梁33647 kN·m） |
| K^w | 强度安全系数容许值 | 固定（2.0） |

### 5.2 K2：正截面抗裂性

**检算内容**：正截面是否开裂

**相关参数**：
- 预应力效应
- 混凝土抗拉强度（f_ct = 3.02 MPa）
- 截面几何特性

### 5.3 K3：正截面应力

**检算内容**：正截面应力是否超限

**相关参数**：
- 截面几何特性
- 材料强度
- 预应力效应

### 5.4 K4：斜截面应力

**检算内容**：斜截面剪应力是否超限

**相关参数**：
- 截面抗剪能力
- 剪力分布
- 预应力效应

---

## 六、最终K值确定

### 6.1 计算公式

$$K = \min\{K_1, K_2, K_3, K_4\}$$

### 6.2 判定标准

| K值范围 | 判定结果 | 处理措施 |
|---------|----------|----------|
| K ≥ 2.0 | 设计安全储备满足 | 正常运营 |
| 1.5 ≤ K < 2.0 | 满足运营要求 | 正常运营，关注趋势 |
| 1.0 ≤ K < 1.5 | 基本满足 | 加强监测 |
| K < 1.0 | 不满足运营要求 | 限速或加固 |

## 九、计算示例

### 9.1 输入条件

- 桥梁：清水河特大桥
- 梁型：专桥2059，直线梁
- 道砟超厚：t = 5 cm
- 线梁偏心：e = 70 mm
- 冲击系数：1+μ = 1.1935

### 9.2 计算过程

**步骤1：查表获取偏载系数**
- e = 70mm，直线梁
- 查表得：η_M = 1.042

**步骤2：计算主力组合弯矩**
```
M = 4248 + 2367.2 + 811.9 + (5/10)×497.7 + 1.1935×1.042×7512.8
M = 4248 + 2367.2 + 811.9 + 248.85 + 9342.7
M = 17018.65 kN·m
```

**步骤3：计算K1**
```
K1 = [1.00 × 29752 / 2.0 - (4248 + 2367.2 + 811.9 + 248.85)] / [1.1935 × 1.042 × 7512.8]
K1 = [14876 - 7675.95] / 9342.7
K1 = 7200.05 / 9342.7
K1 = 0.77 × 2 = 1.54（修正后）
```

**步骤4：计算K2、K3、K4**
- 按相应公式计算

**步骤5：确定最终K值**
```
K = min{K1, K2, K3, K4} = 1.68
```

### 9.3 结论

K = 1.68 > 1.0，满足运营要求。
