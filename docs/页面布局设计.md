## 一、设计概述

### 1.1 设计目标

本系统从"步骤向导"式改为"管理看板"式，核心转变：

| 原设计 | 新设计 |
|--------|--------|
| 流程导向（1→2→3→4） | **桥梁列表为中心** |
| 每次从头开始 | **快速查看历史、一键复算** |
| 单桥梁操作 | **批量管理、筛选统计** |
| 纯计算工具 | **K值管理系统** |

### 1.2 用户角色

- **计算人员**：录入参数、执行计算、保存结果
- **审核人员**：查看计算记录、审核结果、生成报告
- **管理人员**：查看统计、导出数据、配置系统

### 1.3 核心功能

1. 桥梁基础信息管理（从系统同步）
2. K值计算（支持多次历史记录）
3. 计算结果审核流程
4. 检定报告生成与下载
5. 统计分析（可选）

- 曲线半径：曲线梁时必填
- 线梁偏心：数字输入，实时显示查表结果
- 道砟超厚：数字输入
- 冲击系数：默认1.1935，可修改

**损伤修正系数**
- 默认折叠
- 默认值为1.00
- 高级用户可展开修改

**计算结果区**
- 点击"计算"后显示
- 展示K1~K4分项结果
- 突出显示K值（取最小值）

**操作按钮**
- 取消：关闭抽屉
- 保存结果：保存到历史记录
- 保存并生成报告：保存并跳转到报告预览

## 四、页面流转关系

```
                    ┌─────────────┐
                    │   仪表盘    │
                    │  (首页入口)  │
                    └──────┬──────┘
                           │
           ┌───────────────┼───────────────┐
           ▼               ▼               ▼
      ┌─────────┐    ┌─────────┐    ┌─────────┐
      │桥梁列表 │◄──►│统计分析 │    │系统设置 │
      │(核心)   │    │(可选)   │    │        │
      └────┬────┘    └─────────┘    └─────────┘
           │
     ┌─────┴─────┐
     ▼           ▼
┌─────────┐  ┌─────────┐
│桥梁详情 │  │计算抽屉 │
│(历史)   │  │(参数)   │
└────┬────┘  └────┬────┘
     │            │
     └─────┬──────┘
           ▼
      ┌─────────┐
      │报告预览 │
      │(结果)   │
      └─────────┘
```

### 主要操作流程

**1. 首次计算流程**
```
仪表盘/列表 → 选择桥梁 → 开始计算 → 录入参数 → 计算 → 保存 → 报告预览
```

**2. 重新计算流程**
```
列表/详情 → 重新计算 → 修改参数 → 计算 → 保存 → 报告预览
```

**3. 查看历史流程**
```
列表 → 查看详情 → 查看计算历史 → 选择历史记录 → 查看详情/生成报告
```

**4. 审核流程**
```
列表 → 筛选"需复核" → 查看详情 → 审核通过/驳回
```
