## 一、预警设置

### 1.1 到期提醒设置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 提前提醒天数 | 数字输入 | 30 | 首页"本月到期"卡片的计算依据 |
| 紧急阈值天数 | 数字输入 | 7 | 小于此天数显示红色紧急标识 |

**业务逻辑：**
- 到期天数 ≤ 紧急阈值：显示 🔴 紧急
- 紧急阈值 < 到期天数 ≤ 提前提醒天数：显示 🟡 预警
- 到期天数 > 提前提醒天数：不显示在首页卡片

### 1.2 K值预警阈值

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| K值偏低阈值 | 数字输入 | 1.5 | 首页"K值偏低"卡片的筛选阈值 |

**业务逻辑：**
- K值 < 阈值：纳入"K值偏低"统计
- 卡片显示最低K值和数量

### 1.3 首页卡片显示控制

| 配置项 | 类型 | 默认状态 |
|--------|------|----------|
| 总桥梁 | 开关 | 开启 |
| 待计算 | 开关 | 开启 |
| K值偏低 | 开关 | 开启 |
| 本月到期 | 开关 | 开启 |

**业务逻辑：**
- 关闭的卡片不在首页显示
- 至少保留1个卡片开启

## 二、报告模板设置

### 2.1 基础信息配置

| 配置项 | 类型 | 示例 | 用途 |
|--------|------|------|------|
| 单位名称 | 文本输入 | XX铁路局工务段 | 报告抬头单位 |
| 报告编号前缀 | 文本输入 | KZ- | 报告编号自动生成前缀 |
| 审核人默认签名 | 文本输入 | 赵审核 | 报告审核栏默认显示 |
| 计算人默认签名 | 文本输入 | 钱计算 | 报告计算栏默认显示 |

### 2.2 模板预览

点击【预览模板】弹出报告样式预览窗口，展示配置效果。

## 三、用户权限管理

### 3.1 角色定义

| 角色 | 权限范围 |
|------|----------|
| 计算员 | 查看桥梁、执行计算、查看历史 |
| 审核员 | 全部查看 + 审核结果、生成报告 |
| 管理员 | 全部功能 + 系统设置、用户管理 |

### 3.2 用户操作

| 操作 | 功能 |
|------|------|
| 新增用户 | 填写用户名、选择角色、初始密码 |
| 编辑 | 修改角色、重置密码、启用/禁用 |
| 删除 | 删除用户（需确认） |
